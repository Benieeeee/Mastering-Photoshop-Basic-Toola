BRUSH TOOL


The Photoshop brush tool allows the user to paint on the canvas or in the image by clicking the brush tool in the left-hand toolbar. By selecting a color and dragging the tool cursor, the user can draw. The user can also customize the properties of the tool, like its hardness and size, at the top bar option bar. This tool can also create a perfect line by draing the mouse cursor while holding the shift key until the end.
